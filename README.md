# Shopping Cart Web Application

This project is a web-based shopping cart system that allows users to browse products, register/login, and manage items in a cart.

## Features

* User Registration and Login
* Product Catalog
* Add to Cart
* Remove Items from Cart
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in a web browser.
